8 Equivalent Martingale Measures

The idea is that we want to transform the probability measure \(d \bP\) so that the mean of a random process \(z_t\) changes. The transformation allows us to treat assets with positive “risk premium” to be treated as if they were risk free.

Method 1: subtracting means: \(\bar{z} =z - E[z]\)

Method 2: using equivalent measures: Transform \(\bP\) to \(\bQ\) so \(d \bQ = \xi(z_t) d \bP(z_t)\). See Girsanov Theorem

Probability as a measure

  • Use measures to quantify the probability that a random variable falls within a small interval around z
  • \(\int_{-\infty}^{\infty} d \bP (z_t)=1\)
  • \(E(z_t) - \int_{-\infty}^\infty z_t d\bP(z_t)\)
  • \(V(z_t) = E[z_t - E(z_t)]^2 = \int_{-\infty}^\infty[z_t-E(z_t)]^2d\bP z_t\)

Consider a normally distributed random variable \(z_t\) with fixed t and normal density function \(d\bP(z_t)\):

\[ z_t \sim N(0,1) \\ d\bP = \frac{1}{\sqrt{2\pi}}e^{-\frac{1}{2}(z_t)^2} dz_t \] Next, we define the function \(\xi (z_t) = e^{z_t\mu - \frac{1}{2}\mu^2}\) that when we multiply \(\xi(z_t) d\bP(z_t)\) we obtain a new probability:

\[ [d\bP(z_t)][\xi(z_t)] = \frac{1}{\sqrt{2\pi}}e^{-\frac{1}{2}(z_t)^2 + \mu z_t - \frac{1}{2} \mu ^2} dz_t \\ d\bQ(z_t) = \frac{1}{\sqrt{2\pi}}e^{-\frac{1}{2}(z_t - \mu)^2} dz_t \\ \]

Clearly \(d\bQ(z_t)\) is a new probability measure defined as:

\[ d\bQ(z_t) = \xi(z_t) d\bP(z_t) \]

8.1 Radon-Nikodym Derivative

  • \(\frac{d \bQ (z_t)}{d \bP(z_t)} = \xi(z_t)\)
  • The function \(\xi\) allows us to transform between \(\bP\) and \(\bQ\)
  • If both \(\bP\) and \(\bQ\) have the same intervals of \(dz_t\) with non zero probability, then the Radon-Nikodym Derivative exists

8.2 Converting Asset Prices into Martingales

  • Assume that under \(\bP, Y_t \sim N(\rho t, \sigma^2 t)\)
  • Under the true probability distribution \(\bP, E^\bP[e^{-rt} S_t | S_u, u < t]\) is likely not true since, in general, there is a risk premium
  • Switch to the risk-neutral probability \(\bQ\), and then \(E^\bQ[e^{-rt} S_t | S_u, u < t] = e^{-ru} S_u\)

From a SDE viewpoint:

  • Let \(F_t = S_0e^{Y_t}\) where \(dY_t = \mu dt + \sigma dW_t\)
  • \(dF_t = [\mu \frac{\partial F}{\partial Y_t} + \frac{1}{2} \sigma^2_t \frac{\partial ^2 F}{\partial Y_t^2}]dt + \frac{\partial F}{\partial Y_t} \sigma_t dW_t\)
  • All of the partial derivatives shown simply equal \(F_t\)
  • \(dF_t = [\mu + \frac{1}{2} \sigma^2_t]F_t dt + \sigma_t F_t dW_t\)
  • Using Ito’s Lemma gives that the true probability \(\bP\) has a drift coefficient of \((\mu + \frac{1}{2})S_t\)
  • The true probability SDE uses the Wiener process \(W_t\)
  • Setting \(\mu = r - \frac{1}{2} \sigma^2\) gives the PDE \(dS_t = rS_t dt + \sigma S_t dW_t\)
  • Note that the drift parameter is now r, and the Wiener process \(\bar W\) has distribution \(\bQ\)
